Sobre

A aplicação web EbFilmes foi criada com o propósito de que o usuário consiga compartilhar sua opinião com comentário e nota a respeito dos filmes escolhidos. Para o desenvolvimento deste sistema foi utilizado Python, o web framework Django e Bootstrap e CSS no seu layout.
